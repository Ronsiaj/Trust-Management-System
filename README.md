# Trust-Management-System

A full-stack Trust Management System designed to streamline donation handling, visitor engagement, and administrative operations through a centralized digital platform. The system ensures secure donation tracking, efficient donor management, and transparent reporting, helping organizations manage trust activities digitally.

**Features**

Secure donation management system

Donor record tracking and management

Admin dashboard to monitor trust activities

Visitor engagement module

Responsive and user-friendly interface

Real-time data management with database integration

Role-based access for administrators

**Tech Stack**

Frontend

HTML

CSS

JavaScript

ReactJS

Reactstrap

Backend

PHP

Database

MySQL

**System Modules**

**1. Visitor Module**

Handles all interactions for public users visiting the trust website.

Key Functions:

View trust information (About, Vision, Trustees, Programs & Activities, Gallery)

Submit donation requests with beneficiary details and supporting documents

Browse approved donation requests available for funding

**2. Donor Management Module**

Manages donation activities and donor participation.

Key Functions:

Donors select approved requests to support

Support for full or partial donations

Secure payment processing and donation confirmation

Automatic update of donation progress and records

**3. Admin Dashboard Module**

Provides administrative control and monitoring of the system.

Key Functions:

Secure admin login and protected access

Review and approve/reject donation requests

Manage donor records and donation transactions

Monitor trust activities through analytics and summary reports

**Installation**

1. Clone the repository:

git clone https://github.com/your-username/trust-management-system.git

2. Navigate to the project directory:

cd trust-management-system

3. Install frontend dependencies:

npm install

4. Start the React development server:

npm start

Configure backend server (PHP) and connect to MySQL database.

**Database Setup**

1. Create a MySQL database.

2. Import the provided SQL file.

3. Update database configuration in the backend.
